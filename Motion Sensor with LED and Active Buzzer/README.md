**MOTION SENSOR WITH LED AND ACTIVE BUZZER**

Motion sensor senses movement based on IR light. When it senses movement, 
the LED will light up, and active buzzer will start making sound. Also there are output sentences shown in the
Serial Monitor. "Nobody here" when the sensor detects no movement, and "There he is!" when the sensor detects movement.
I don´t know why, but there is some sort of bug, probably on my side, that the sensor has like a 3 second delay, 
and sometimes it doesn´t even senses movement.
