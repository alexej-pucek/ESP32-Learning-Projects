**RGB LED WITH CHANGING VALUES BASED ON POTENTIOMETER INPUT**

This is a quick project using RGB LED. You map each colour of the RGB (Red, Green, Blue) to potentiometers, 
and than based on the potentiometer input, you can change the RGB values of the LED,
and create huge amount of colours.

TIME: 20 min + 10 min debugging.

DIFFICULTY: 2/10.

LEARNING VALUE: 5/10.


DATE: 11.7.2026.
