**Humidity and Temperature Monitor Using DHT11**

I made a simple humidity and temperature monitor using the DHT11 module. 
I used the DHTesp library (https://github.com/beegee-tokyo/DHTesp) which worked just fine with my ESP32.
I had to use F - F cables, because for some reason my Breadboard is bugging.
