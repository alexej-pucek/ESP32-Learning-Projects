**PHOTORESISTOR WITH LED**

Today I made a project, where LED would light up if it got dark in the room. I made it in two phases.
First phase was to wire up the photoresistor, and print its values to serial monitor, 
and check what is the value when the place is dark. For me it was around 2000.
Second phase was to wire up the LED, and set it to light up only when the value is higher than 2000.
